# Database schema

## `travelers`

Описание: отдельные путешественники.

Поля:

- `traveler_id` — идентификатор путешественника, например `T-0001`.
- `full_name` — имя путешественника.
- `age` — возраст.
- `citizenship` — гражданство.
- `home_airport` — базовый аэропорт.
- `loyalty_program` — программа лояльности, если есть.
- `notes` — текстовые комментарии и мягкие предпочтения.

## `traveler_preferences`

Описание: структурированные предпочтения отдельных путешественников.

Поля:

- `traveler_id` — идентификатор путешественника.
- `preference_type` — тип предпочтения.
- `preference_value` — значение предпочтения.
- `comment` — пояснение к предпочтению.

## `travel_groups`

Описание: параметры поездки для группы.

Поля:

- `group_id` — идентификатор группы.
- `origin_city` — город вылета.
- `destination` — направление.
- `start_date` — дата начала поездки.
- `end_date` — дата окончания поездки.
- `budget_rub` — бюджет в рублях.
- `group_comment` — общий комментарий по группе.

## `group_members`

Описание: связь групп и путешественников.

Поля:

- `group_id` — идентификатор группы.
- `traveler_id` — идентификатор путешественника.
- `role_in_group` — роль в группе: например `parent`, `child`.

## `flights`

Описание: доступные перелёты.

Поля:

- `flight_id` — идентификатор перелёта.
- `origin_city` — город вылета.
- `destination` — направление.
- `price_rub` — цена перелёта.
- `baggage_included` — включён ли багаж: `1` или `0`.
- `stops` — количество пересадок.
- `departure_time` — время вылета.
- `arrival_time` — время прилёта.
- `fare_type` — тип тарифа.
- `notes` — пояснение к перелёту.

## `hotels`

Описание: доступные отели.

Поля:

- `hotel_id` — идентификатор отеля.
- `destination` — направление.
- `stars` — количество звёзд.
- `price_per_night_rub` — цена за ночь.
- `breakfast_included` — включён ли завтрак: `1` или `0`.
- `free_cancellation` — есть ли бесплатная отмена: `1` или `0`.
- `rating` — рейтинг отеля.
- `notes` — пояснение к отелю.

## `tours`

Описание: пакетные туры.

Поля:

- `tour_id` — идентификатор тура.
- `destination` — направление.
- `total_price_rub` — итоговая цена тура.
- `includes_flight` — включает ли перелёт.
- `includes_transfer` — включает ли трансфер.
- `hotel_id` — связанный отель.
- `notes` — пояснение к туру.
