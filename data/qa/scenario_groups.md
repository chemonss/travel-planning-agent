# Сценарные группы путешественников

## G-0001
Семья из двух взрослых и ребёнка. Ключевые ограничения: без ночного прилёта, завтрак желателен, бюджет ограничен.

## G-0002
Пара для пляжного отдыха. Ключевые предпочтения: спокойный отель, минимум пересадок, допускается пакетный тур.

## G-0003
Один путешественник. Важны багаж, комфортный график и отсутствие слишком раннего вылета.

## G-0004
Бюджетная поездка, допустим пакетный тур. Используется для сценариев изменения бюджета.

## G-0005
Пара с конфликтующими требованиями: только прямой перелёт, только 5* и ограниченный бюджет. Используется для кейсов clarification/rejection.

## G-0006
Группа с ребёнком и визовым риском. Используется для сценариев обязательной эскалации.
